======================
 The ``MyList`` Class
======================

Using ``MyList``
----------------

Inherits ``list`` class, with additional ``print_sorted()`` instance
method that prints the list sorted without affecting the list.

Setup
=====

::

	>>> MyList = __import__('1-my_list').MyList

Create List
===========

::

	>>> my_list = MyList()

Append Elements
===============

::

	>>> my_list.append(3)
	>>> my_list.append(33)
	>>> my_list.append(12)
	>>> my_list.append(1)
	>>> my_list.append(4)

Print List
==========

::

	>>> print(my_list)
	[3, 33, 12, 1, 4]
	>>> my_list.print_sorted()
	[1, 3, 4, 12, 33]
	>>> print(my_list)
	[3, 33, 12, 1, 4]
